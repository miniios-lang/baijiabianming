# 百家辨鸣 · 抖音视觉形象系统

> 百家思想 · 万千共鸣

心灵鸡汤 + 情绪共鸣 + 文案表达类抖音账号的完整视觉形象设计，采用**国潮古风**风格，以**墨色 + 暖金**为主色调。

## 在线预览

[打开主页效果预览](https://miniios-lang.github.io/baijiabianming/)

在手机或电脑浏览器中打开上方链接，即可查看模拟抖音个人主页效果，包含头像、横幅、4 种封面模板，以及素材下载功能。

## 使用方式

1. 打开 [在线预览页](https://miniios-lang.github.io/baijiabianming/)
2. 滑到底部「素材下载」区域
3. 点击对应按钮即可下载 PNG 文件
4. 点击「全部下载 PNG」可一次性导出全部素材

## 设计风格

- **色调**：墨色系（#1a1a1a ~ #2d2d2d）+ 暖金色（#c9a96e ~ #d4b878）
- **字体**：思源宋体（标题/展示）+ 思源黑体（正文/可读性）
- **装饰元素**：云纹、山水剪影、印章、竹枝、莲花
- **外观切换**：页面内置 4 种风格预设（墨夜 / 亮色 / 暖金 / 朱砂），可通过右侧面板微调
